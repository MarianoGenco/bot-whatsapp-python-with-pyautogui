# WHATSAPP BOT WITH PYAUTOGUI AND SPREADSHEET
You can send messages, with a spreadsheet. 

# How install

1) Clone or download repositorie
2) Open with visual studio code the folder
3) Type in terminal <code>virtualenv venv</code>
4) Type then <code>.\venv\Scripts\activate</code>
5) Type then <code>pip install -r requirements.txt</code>
6) Go for it <code>python bot.py</code>
